# ⚡ Início Rápido - CodAtende Chat Local

## 🎯 Para começar agora mesmo:

### 1. Primeira vez usando?
```bash
./instalacao-local/primeiro-uso.sh
```

### 2. Já configurou antes?
```bash
./instalacao-local/deploy-local.sh
```

### 3. Algo deu errado?
```bash
./instalacao-local/diagnostico.sh
```

---

## 🌐 Acesse:
- **Frontend**: http://localhost:3000
- **Backend**: http://localhost:8080

---

## 🛠️ Comandos Básicos:

```bash
# Parar projeto
docker-compose down

# Ver logs
docker-compose logs -f

# Reiniciar
docker-compose restart

# Limpar tudo
docker-compose down -v
```

---

## 📋 Pré-requisitos:
- Docker instalado
- Docker Compose instalado
- Portas 3000, 8080, 5432, 6379 livres

---

## 🆘 Problemas?
1. Execute: `./instalacao-local/diagnostico.sh`
2. Leia: `DICAS.md`
3. Veja logs: `docker-compose logs -f`
